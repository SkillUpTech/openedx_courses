Concept: String Comparison

String Comparisons

  • Strings can be equal == or unequal !=

  • Strings can be greater than > or less than <

  • alphabetically "A" is less than "B"

  • lower case "a" is greater than upper case "A"

Example

# review and run code
"hello" < "Hello"

# review and run code
"Aardvark" > "Zebra"

# review and run code
'student' != 'Student'

# review and run code
print("'student' >= 'Student' is", 'student' >= 'Student')
print("'student' != 'Student' is", 'student' != 'Student')

# review and run code
"Hello " + "World!" == "Hello World!"
