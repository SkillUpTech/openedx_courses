Take Advantage of the CS Fundamentals Learning Curriculum

This course is part of the Computer Science Fundamentals Curriculum from Microsoft Learning. Our curriculum consists of a cohesive set of courses that teach the fundamentals of computing and programming.

As a learner, you'll develop fundamental computing skills starting with the basics of computer science and computational thinking through a working knowledge of how computers operate, basic programming skills and professional programming practices.

By completing this curriculum, you'll be prepared to continue on to intermediate and advanced courses offered by Microsoft Learning, where you can dive deeper into each of the technologies you've been exposed to in this coursework.

How to Approach This Curriculum

Each course builds on the courses before it, and learners are expected to master the subjects at each stop before proceeding to the next one. As you complete these courses, you will progress in skill level from Self-Starter to Apprentice to Intern and ultimately to Entry-Level Software Developer.

The CS Fundamentals Courses:

Logic and Computational Thinking : Build a solid foundation for programming by learning basic logic and exploring how logic forms the foundation of computer programs.

Introduction to Python: Absolute Beginner: In this course that’s perfect for true beginners, learn Python basics and start coding right away.

Introduction to Python: Fundamentals : Build on your basic Python skills and dig into data structure basics.

Designing a Technical Solution: Learn the process of designing and prototyping web applications to solve real problems.

Building Interactive Prototypes using JavaScript : Learn how you can use JavaScript, the programming language of the Web, to build interactive webpages.

Building Functional Prototypes using Node.js : Learn the basics of back-end web development with Node.js.

Learn to Program in Java : Get started on the path to becoming a software engineer by learning core coding skills in Java—one of the most popular programming languages.

Object-Oriented Programming in Java : One of the most important skills in software development is designing code organization. In this Java course, learn how to apply industry-standard practices as you structure your software.

Algorithms and Data Structures : Learn to analyze algorithms effectively and implement advanced data structures to develop efficient software programs.

Writing Professional Code : Getting code to run and compile is, as the best programmers know, only a small part of writing great programs. Build upon your programming skills as you learn to write readable, secure, maintainable and well-tested code.

Introduction to Design Thinking : Explore the fundamentals of how humans and machines interact and learn how to design effective user experiences.

(Coming Soon) Web Accessibility Fundamentals : Learn the best practices and guidelines needed to create accessible web applications and empower all of your users.

Developing International Software : Learn how to design and develop software for international use from experts who’ve helped create some of the world’s most successful software.

Learning Outcomes

Our goal is that a learner who has completed our Fundamentals curriculum can function in a job environment as an entry-level software developer once the concepts and practice exercises in these courses have been mastered. From there, you can take advantage of the many other courses Microsoft Learning offers in specific focus areas, including our Professional Programs in Front-End Web Development and Data Science.

We're glad you're here and we encourage you to complete the courses in this curriculum in the prescribed order. You're on your way to mastering the fundamentals of Computer Science!
