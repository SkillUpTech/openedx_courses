Project File Downloads

Each Section has an associated .ipynb (Jupyter Notebook) where samples can be run, and where practice tasks and code assignments can be completed. The files are available on this page.

Course Materials for Introduction to Python Unit 1 on edX also come in a Jupyter Notebook format (.ipynb files) that enable the student to:

  • run sample code
  • code and run the practice Tasks
  • code and run the unit practices
  • code and run the code assignments (4 end of unit and 1 final)

Notebooks can be cloned or uploaded using Azure Notebooks (https://aka.ms/edx-dev236x-az) which allows all work to be done and saved via a modern web browser and internet connection.
